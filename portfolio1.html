Home Lab

A home server system running (mainly) open-source software to host personal files, machine learning-based photo storage, and centralized control over every devices using the open-source Home Assistant project. I retrofitted the house by hand-building the thermostat, energy sensor, and garage door switch using wireless ESP-8266 microprocessor boards running ESPHome.

Host Machine: TrueNAS Scale, Ubuntu Server VM

Services: Home Assistant, FileRun, Immich, NGINX Proxy Manager, Portainer
