Home Assistant

A home automation system that integrates every smart device in the house, including light switches, thermostats, garage door, and home energy sensors. Centralized control over every devices using the open-source Home Assistant project. I retrofitted the house by hand-building the thermostat, energy sensor, and garage door switch using wireless ESP-8266 microprocessor boards soldered onto various components, programmed with ESPHome to integrate natively with the Home Assistant API. And of course, I can automate tasks and devices according to the time of day and have one event influence another device; for example, using a vehicle location tracker, I am able to open the garage door automatically upon arriving home. Runs on an Ubuntu Server virtual machine on an old server pc from Free Geek volunteering.
